# MORE.TECH HACK 2025 / Трек - LC/NC

Сервис для автоматического и ручного планирования посещений офиса и отпусков сотрудников с учетом вместимости офиса, пересечений и ограничений по рабочим местам. Проект разработан в рамках хакатона [MORE.TECH HACK 2025](https://moretech.vtb.ru/) с помощью платформы FIS-Platform для заказчика, который столкнулся с беспорядком при управлении офисной нагрузкой и графиками персонала

---
### Реализованный функционал:
- Генерация рабочего графика
- Возможность выгрузки графика в .csv / .xlsx
- Базовая аналатика, которая позволяет следить за загруженностью офиса и сотрудников
- Загрузка производственного календаря
- Гибкая настройка сотрудников, офисов, отпусков, мероприятий и др.

### Технологии и инструменты:
- [FIS-Platform](https://fisgroup.ru/products/fis-platform/)
- JavaScript

### Результат:
- Планирование графика на следующий месяц занимает меньше минуты, вместо 10-15 часов
- Пересечение легко заметны, сервисом просто пользоваться
- Каждому есть место в офисе
- С аналитикой понятно, как загружены офис и сотрудники
